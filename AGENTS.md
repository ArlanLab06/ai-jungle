This project is called “Into the AI Jungle”.

It is an interactive educational website about “Artificial Intelligence: Friend or Threat?” for an English course final project.

Style:
- cinematic dark jungle
- subtle technology
- warm gold trail light
- soft emerald accents
- dark glass cards
- premium atmosphere
- no heavy neon cyberpunk dashboard
- no overloaded UI

Core flow:
Name input → techno-cat Byte → jungle journey → 4 zones → choices → score → final verdict.

Required zones:
1. Education
2. Healthcare
3. Business
4. Daily Life

Required features:
- user enters name
- name saved in localStorage
- floating techno-cat Byte speaks with predefined messages
- responsible/dangerous choices
- Friend Score, Threat Score, Responsibility Score
- final verdict
- Presentation Mode
- responsive design

Stack:
Next.js + TypeScript + Tailwind CSS + Framer Motion.
