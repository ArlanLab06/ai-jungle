# Into the AI Jungle

Interactive website for the English final project "Artificial Intelligence: Friend or Threat?".

The project guides users through a cinematic jungle journey where they explore how AI affects education, healthcare, business, and daily life. Users make choices, receive feedback from Byte the techno-cat guide, and get a final verdict: AI is a friend, a threat, or both.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## Main features

- Name input and personalized journey
- Floating techno-cat guide named Byte
- Four AI zones
- Responsible vs dangerous AI choices
- Friend, Threat, and Responsibility scores
- Final verdict
- Presentation Mode
- Responsive design

## Core message

AI is a friend when it supports people. AI is a threat when it replaces human thinking, privacy, honesty, and responsibility.
